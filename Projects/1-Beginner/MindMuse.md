## Description
MindMuse is a simple journal app that provides daily creativity prompts and helps users build a consistent creative habit through writing, drawing, or idea logging.

### Purpose
To boost creativity and mindfulness through short, daily exercises.

### Resources Needed
No external APIs required. Prompts can be stored locally or fetched from a static JSON file.

---

## User Stories
- [ ] User can view a new creative prompt each day.
- [ ] User can write or sketch their response.
- [ ] User can browse previous entries by date.
- [ ] User can mark favorite prompts.
- [ ] User can back up data locally.
