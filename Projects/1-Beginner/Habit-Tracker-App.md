# Habit Tracker App

**Tier:** 1-Beginner

Habit trackers are powerful tools for personal development, helping users build and maintain positive routines. In this app, you'll create a simple habit tracker that allows users to log daily habits, view streaks, and receive reminders. This project is great for understanding state management, data persistence, and basic UI interactions.

The design is up to youâ€”get creative with colors, icons, and layouts! You might draw inspiration from popular habit-tracking apps on your phone to understand core functionality and user experience.

### Constraints

- You may not use external APIs for habit data storage (use local storage or a simple file-based system instead).
- Keep the app focused on core habit tracking without complex analytics.

## User Stories

- [ ] User can see a dashboard displaying all added habits with their current streak and completion status for the day.
- [ ] User can add a new habit by entering a name (e.g., "Drink water") and optional description.
- [ ] User can mark a habit as completed for the current day by clicking a checkbox or button.
- [ ] User can view a streak counter for each habit (e.g., "Completed 5 days in a row").
- [ ] User can delete or edit an existing habit.
- [ ] User can receive daily reminders (via browser notifications or simple alerts) for incomplete habits.
- [ ] User can view a history of habit completions over the past week or month.

## Bonus features

- [ ] User can set custom goals for habits (e.g., "Complete 5 times a week").
- [ ] User can categorize habits (e.g., Health, Productivity, Learning).
- [ ] User can export habit data as a CSV or JSON file.
- [ ] User can view a calendar heatmap showing habit completion patterns.
- [ ] User can add notes or reflections for each habit completion.
