# Screen resolutions checker

**Tier:** 1-Beginner

Mobiles, laptops, computer monitors come in various sizes and support various resolutions, and it is important to make sure that you know your preferred resolutions. That is what this project is for. The code also adapts pixels for resolution measurements.

- Code editors and Mozilla Developer Network (https://developer.mozilla.org/en-US/) 
 

## User Stories

-   [ ] User can use the app to find preferred sizes for different online browser tabs by finding pixel sizes in our app and then having the browser tab open and ctrl+tabbing to find the sizes.
-   [ ] User can test their own websites in different resolutions and orientations by finding the pixel sizes and then ctrl+tabbing to their own app and configuring their setups.
-   [ ] User can find their available screen space and current window size.
-   [ ] User can copy individual metrics easily using the count in pixels and its text format.

        
## Bonus features

-   [ ] User can find orientation of page, whether or not it is a landscape, portrait, or square format, based on the pixel widths and heights.
-   [ ] Users can fullscreen the tab to find their screen resolution and their maximum window resolution.
-   [ ] Users can also add their own metrics by quickly adapting the measurements in the code with their own metrics and applying basic conversions.
