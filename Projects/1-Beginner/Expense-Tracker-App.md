# Expense Tracker App

**Tier:** 1-Beginner

Expense trackers are essential tools for personal finance management, helping users monitor spending, stay within budgets, and develop better money habits. In this app, you'll create a simple expense tracker that allows users to log expenses, categorize spending, and view summaries. This project is great for understanding CRUD operations, data persistence, and basic financial calculations.

The design is up to youâ€”get creative with colors, charts, and layouts! You might draw inspiration from popular budgeting apps on your phone to understand core functionality and user experience.

### Constraints

- You may not use external APIs for expense data storage (use local storage or a simple file-based system instead).
- Keep the app focused on expense tracking without complex investment or banking features.

## User Stories

- [ ] User can see a dashboard displaying total expenses, remaining budget, and a summary for the current month.
- [ ] User can add a new expense by entering amount, category (e.g., Food, Transport, Entertainment), date, and optional description.
- [ ] User can view a list of all expenses with details (amount, category, date, description).
- [ ] User can edit or delete an existing expense.
- [ ] User can set a monthly budget limit and see how much is remaining.
- [ ] User can filter expenses by date range (e.g., last week, last month, custom range).
- [ ] User can filter expenses by category to see spending patterns.
- [ ] User can see a warning notification when approaching or exceeding the budget limit.

## Bonus features

- [ ] User can view expense breakdown by category using a pie chart or bar graph.
- [ ] User can add custom expense categories with color coding.
- [ ] User can export expense data as a CSV or PDF file.
- [ ] User can set up recurring expenses (e.g., monthly rent, subscriptions).
- [ ] User can switch between different currencies with conversion rates.
- [ ] User can view spending trends over multiple months with line charts.
- [ ] User can add receipt photos to expenses.
- [ ] User can set category-specific budget limits (e.g., $200 for Food, $100 for Entertainment).
- [ ] User can search expenses by description or amount.
- [ ] User can switch between light and dark themes.
