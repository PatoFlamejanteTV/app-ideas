# Tip Calculator App

## 1. Objective

The Tip Calculator is a simple application used to calculate the tip amount for a bill and determine the total amount each person needs to pay. This is a practical tool for quickly splitting a check among friends at a restaurant.

This app will help you practice fundamental UI development, user input handling, and basic mathematical calculations.

## 2. User Stories

-   [ ] User can enter the total cost of the bill.
-   [ ] User can select a tip percentage from a predefined list of options (e.g., 10%, 15%, 18%, 20%).
-   [ ] User can also input a custom tip percentage.
-   [ ] User can enter the number of people to split the bill with.
-   [ ] The application should calculate and display the tip amount per person.
-   [ ] The application should calculate and display the total amount per person (bill + tip).
-   [ ] The input fields should be validated to accept only numbers.

## 3. Bonus Features

-   [ ] Add a "Round Up" button that rounds the total amount per person to the nearest whole number.
-   [ ] Display a summary of the calculation: Total Bill, Total Tip, and Total Amount.
-   [ ] Add a "Reset" button to clear all input fields and start a new calculation.
-   [ ] Implement a theme switcher (e.g., Light/Dark mode) for the UI.
