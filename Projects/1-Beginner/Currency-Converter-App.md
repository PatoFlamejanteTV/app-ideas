# Currency Converter

**Tier:** 1-Beginner

A Currency Converter app allows users to convert amounts from one currency to another using real-time or static exchange rates. This project helps you understand API integration, handling user input, and performing calculations based on dynamic data.

The styling is up to you, so use your imagination and get creative! This is a practical tool that can help you learn about working with external APIs and handling real-world data.

## User Stories

-   [ ] User can enter an amount in a numeric input field
-   [ ] User can select a source currency from a dropdown list
-   [ ] User can select a target currency from a dropdown list
-   [ ] User can click a 'Convert' button to perform the conversion
-   [ ] User can see the converted amount displayed on the screen
-   [ ] User can see the current exchange rate being used for the conversion
-   [ ] User can swap the source and target currencies with a single click/button

## Bonus features

-   [ ] User can see a list of popular currencies with their exchange rates relative to a base currency
-   [ ] User can see historical exchange rate data displayed in a chart
-   [ ] User can select a date to see historical exchange rates
-   [ ] User can see the last updated timestamp for exchange rates
-   [ ] User can convert multiple currencies at once (compare rates)
-   [ ] App works offline using cached exchange rates with a notice that rates may not be current
