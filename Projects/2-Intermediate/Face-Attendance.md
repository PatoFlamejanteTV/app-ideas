# Face Attendance app

**Tier:** 2-Intermediate

The Attendance management system that captures face image and mark the attendance and timings

## User Stories

-   [ ] User can add new character/user to mark the attendance
-   [ ] User can see the date and timings of the attendance
-   [ ] User can set the threshold ( similarity quotient ) for the comparision
-   [ ] User can update the existing face image

## Bonus features

-   [ ] User can download particular date's attendance
-   [ ] User can get email of the list of attendees everyday
-   [ ] User can email the individual attendee's monthly attendance report.
-   [ ] User can set a minimum attendance for the month to track eligibility of attendees
