# Wordify - Dictionary API

**Tier:** 2-Intermediate

A dictionary application that fetches word definitions, phonetics, parts of speech, and related images using external APIs. This project helps developers practice API integration while building a useful reference tool.

The app allows users to search for any word and displays comprehensive information including pronunciation, definitions, usage examples, and a relevant image to enhance understanding.

## User Stories

-   [ ] User can enter a word in a search input field
-   [ ] User can click a search button or press Enter to look up the word
-   [ ] User can see the word's pronunciation displayed in phonetic notation
-   [ ] User can hear the word's pronunciation by clicking a play button
-   [ ] User can see multiple definitions if the word has different meanings
-   [ ] User can see the part of speech (noun, verb, adjective, etc.) for each definition
-   [ ] User can see usage examples for each definition
-   [ ] User can see a relevant image related to the searched word
-   [ ] User can see an error message if the word is not found
-   [ ] User can see a loading indicator while the API request is being processed

## Bonus features

-   [ ] User can see word synonyms and antonyms
-   [ ] User can see the word's etymology (origin and history)
-   [ ] User can save favorite words to a personal dictionary
-   [ ] User can see a random word of the day feature
-   [ ] User can share word definitions on social media
-   [ ] User can see recently searched words in a history list
-   [ ] User can switch between different languages for definitions
-   [ ] User can see word frequency and difficulty level
